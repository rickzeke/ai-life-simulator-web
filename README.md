# AI生活模拟器-职场修炼 - 产品规划

## 产品定位
AI生活模拟器的职场篇章，一款以职场生存为题材的文字冒险游戏。玩家将扮演一名职场新人，从投递简历到职场晋升，体验真实的职场生态。游戏强调"选择即命运"——每一次对话、每一个决策都可能影响你的职业轨迹。

## 目标用户
- 即将步入职场的大学生
- 初入职场的年轻上班族
- 对职场题材感兴趣的游戏玩家
- 希望在娱乐中了解职场规则的用户

## 核心功能

### 场景一：租房奇遇（已有）
- 房东互动、房源选择、签约决策
- 6种不同结局

### 场景二：职场修炼（新增）
- **公司选择**：大厂/外企/创业公司/国企，每种公司有不同文化
- **面试模拟**：AI驱动的多轮面试对话，影响入职起点
- **职场日常**：处理同事关系、应对上级、加班博弈
- **技能养成**：沟通/技术/管理/政治 四大技能树
- **晋升之路**：从新人到总监，多条职业路径
- **结局系统**：10+种职场结局（被裁、晋升、创业、躺平等）

### 通用功能
- 本地存储进度保存
- 成就系统
- 分享结果

## 变现设计
- 广告变现（激励视频广告用于复活/跳过）
- 皮肤/主题付费
- 未来可扩展：付费剧情DLC

## 技术方案
- 纯前端HTML/CSS/JS，无后端依赖
- localStorage存档
- 可扩展的对话树数据结构
- 响应式设计，移动端适配

## 开发计划
- Phase 1: 基础框架 + 场景选择器
- Phase 2: 职场剧情内容开发
- Phase 3: 技能系统 + 成就系统完善
- Phase 4: UI美化 + 细节打磨
